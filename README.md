<strong>What's G-OS?</strong><br>
G-OS is a teeny tiny OS that is entirely made by JS, HTML, and CSS, and G-OS also allows you passwords too!

<strong>How to get in when I forget my password?</strong><br>
Read "HOW TO GET IN IF PASSWORD IS FORGOTTEN.md" for more information.


<strong>How to download G-OS?</strong><br>
You don't entirely <strong>download</strong> it, but you can upload it, and double press one of the files that have "G-OS" and a number beside it, all of them are particularly the same.
